Installation

  • Install TeX Live [ TeX Live ].
  • Install latexmk via TeX Live Manager. See [CTAN].
  • Download Visual Studio Code [here ].
  • Add a VScode Extension callded "LaTeX Workshop".

Setup code snippets

Go to "File" >> "Preference" >> "Configure User Snippets" and open "latex.json" file. This file looks like below by default. 
         {
   // Place your snippets for latex here. 
   // ・・・
   // same ids are connected.
   // Example:
   // "Print to console": {
   // 	"prefix": "log",
   // 	"body": [
   // 		"console.log('$1');",
   // 		"$2"
   // 	],
   // 	"description": "Log output to console"
   // }
}
Add your code snippets in the latex.json. Here is an example. 
         
"article": {
   "prefix": "article",
   "body": [
      "\\documentclass[${1:a4paper,11pt}]{${2:article}}",
      "",
      "",
      "\\usepackage{amsmath, amssymb, amsfonts, latexsym, mathtools}",
      "\\usepackage[amsmath,amsthm,thmmarks]{ntheorem}",
      "\\usepackage{bm}",
      "\\usepackage[dvipdfmx]{graphicx}",
      "${3}",
      "",
      "\\begin{document}",
      "",
      "\\title{${4: title }}",
      "\\author{${5: author }}",
      "\\date{${6:\\today}}",
      "\\maketitle",
      "Let's write an article ...",
      "",
      "$0",
      "",
      "\\begin{thebibliography}{99}",
      "\\bibitem{textbook} author, \\emph{title of a book}, publisher, year",
      "\\end{thebibliography}",
      "\\end{document}"
   ],
   "description": "template for an article"
}
Create new tex file and type the prefix (in this case "article") and Voilà! You can write a latex code anytime. 
         
\documentclass[a4paper,11pt]{article}


\usepackage{amsmath, amssymb, amsfonts, latexsym, mathtools}
\usepackage[amsmath,amsthm,thmmarks]{ntheorem}
\usepackage{bm}
\usepackage[dvipdfmx]{graphicx}


\begin{document}

\title{ title }
\author{ author }
\date{\today}
\maketitle
Let's write an article ...



\begin{thebibliography}{99}
\bibitem{textbook} author, \emph{title of a book}, publisher, year
\end{thebibliography}
\end{document}
